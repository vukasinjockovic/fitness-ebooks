# Drop Sets

**Type:** tactic
**Status:** solid
**Last Updated:** 2026-02-03
**Aliases:** Strip Sets, Descending Sets, Running the Rack

## Summary

Drop sets are John Meadows' self-described "favorite technique" - a simple but effective intensification method where you perform reps to failure, immediately reduce weight, and continue for additional reps. The technique can be repeated 3-4 times in succession, creating extreme metabolic stress and muscle pump.

## How to Perform

1. Perform target reps to failure
2. Immediately reduce weight (no rest)
3. Continue repping to failure
4. Repeat 2-4 times total

## Key Implementation Detail

> "Learn how much to drop the weight. This will depend heavily on how hard you pushed the previous set."
— Ch. 2

**Planning drops:**
- If you went to COMPLETE failure: larger drop needed
- If you left 1-2 reps in tank: smaller drop works
- Sequence plates on bar for quick drops
- Common sense, but be aware when planning

## Example Application

Smith machine incline press:
- Set 1: 185 lbs x failure → drop to 135
- Set 2: 135 lbs x failure → drop to 95
- Set 3: 95 lbs x failure (optional 4th drop)

## When to Use

- **Phase 3:** Supra-Maximal Pump exercises (primary use)

- Machine exercises work best (easy weight changes)
