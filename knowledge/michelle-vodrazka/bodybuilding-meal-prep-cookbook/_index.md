# The Bodybuilding Meal Prep Cookbook

**Author:** Michelle Vodrazka
**Publisher:** Rockridge Press
**Published:** 2019
**Subtitle:** Macro-Friendly Meals to Prepare, Grab, and Go

## Overview

A progressive 6-week meal prep program designed for bodybuilders and fitness enthusiasts. Unlike other cookbooks that just provide recipes, this book teaches you how to meal prep efficiently with a structured week-by-week program that gradually increases in complexity.

## Key Features

### Recipe Format
Each recipe includes:
- Macro percentages (% Protein / % Carbs / % Fat)
- Full nutrition per serving (calories, fat, protein, carbs, fiber, sugar, sodium)
- Dietary tags (Dairy-Free, Gluten-Free, Nut-Free, Vegan, Vegetarian)
- Storage and reheating instructions
- Substitution tips

### Meal Prep Plans Include
- 5-day meal schedule
- Complete shopping list by category
- Equipment checklist
- Step-by-step prep instructions

## Approach

The book emphasizes:
1. **Progressive complexity** - Start simple, build skills over 6 weeks
2. **Batch cooking** - Efficient prep sessions
3. **Macro tracking** - Every recipe shows macro percentages
4. **Flexibility** - Dietary tag filters for restrictions
5. **Storage optimization** - Refrigeration and freezing guidelines

## Author

Michelle Vodrazka is a certified personal trainer, fitness nutrition specialist, and food blogger. She specializes in meal prep strategies for busy fitness enthusiasts.
