# Tactical Barbell Operator Template
**Type:** framework
**Status:** solid
**Last Updated:** 2026-02-03
**Source:** Tactical Barbell: Physical Preparation for Law Enforcement (2015) by K. Black
**Aliases:** Operator, TB Operator, Operator Program

## Summary
The Operator template is a strength training program from the Tactical Barbell system that uses percentage-based progressive loading with a minimal exercise cluster. It's designed for tactical athletes who need to develop strength while maintaining conditioning. In PPLE, Operator is used during Block I (Base Building).

## Exercise Cluster
The Operator template uses only three exercises:
1. **Barbell Squat**
2. **Barbell Bench Press**
3. **Bodyweight Pull-ups**

Plus supplementary deadlift work during certain sessions.

> "Those are the only three exercises you're going to use consistently during your strength training sessions."
— Source: Chapter 4, "Strength"

## Training Frequency
- **Block I:** 3 strength sessions per week (all exercises each session)
- Uses percentage-based progression from tested 1RM
— Source: Chapter 4

## Progression Model
1. Test one-repetition maximums (or calculate from reps)
2. Lift prescribed percentages for sets/reps each session
3. Some weeks light and easy, others heavy and difficult
4. Re-test maximums after training block
5. Use new maximums for next cycle
— Source: Chapter 4

## Example Week (Block I, Week 1)
| Session | Squat | Bench | Pull-ups |
|---------|-------|-------|----------|
| Session 1 | 3x5 @ 70% | 3x5 @ 70% | 3 sets @ 70% max |
| Session 3 | 3x5 @ 70% | 3x5 @ 70% | 3 sets @ 70% max |
| Session 5 | 3x5 @ 70% | 3x5 @ 70% | 3 sets @ 70% max |
— Source: Chapter 8

## Key Principles
1. **Rest Interval Rule:** Minimum 2 minutes between sets
2. **Avoid Muscle Failure:** Stay fresh for all sets
3. **No Extra Work:** Do exactly what's prescribed, no more
4. **Percentage-Based:** Individual progression based on YOUR maxes

> "There is no generic advice in this program such as 'squat 150lbs today for 3 x 5'"
— Source: Chapter 2
