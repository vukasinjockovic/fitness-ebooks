# Set Point Theory
**Type:** concept
**Status:** solid
**Last Updated:** 2026-02-03
**Aliases:** Body Weight Set Point, Defended Weight, Settling Point

## Summary

The hypothesis that the body defends a particular body weight or body fat level through homeostatic mechanisms. When weight deviates from this set point (through dieting or overeating), the body activates compensatory mechanisms to return to the defended weight. This explains why weight regain after dieting is so common and why maintaining weight loss requires ongoing effort.

## How Set Point Defense Works

### When Below Set Point (Dieting):
- Increased hunger signals (ghrelin up)
- Decreased satiety signals (leptin down)
- Reduced metabolic rate
- Lower NEAT and spontaneous activity
- Enhanced food reward sensitivity
- Psychological preoccupation with food
— Source: Fat Loss Forever, Ch.3

### When Above Set Point (Overfeeding):
- Decreased appetite
- Increased NEAT
- Higher metabolic rate
- Less efficient energy storage
- (Though this direction is weaker than defense against loss)
— Source: Fat Loss Forever, Ch.3

## The Asymmetry Problem

The body defends more strongly against weight LOSS than weight GAIN:
- Evolutionary pressure: famine more common than abundance
- No survival advantage to preventing obesity
- This is why it's easier to gain than to lose
— Source: Fat Loss Forever, Ch.1, Ch.3

## Can Set Point Change?

**Factors that may lower set point:**
- Exercise (especially resistance training)
- Long-term weight maintenance
- Time at new weight
— Source: Fat Loss Forever, Ch.9

**Factors that may raise set point:**
- Repeated yo-yo dieting
- Prolonged overfeeding
- Aging
- Certain medications
— Source: Fat Loss Forever, Ch.1

## Key Insight

> "Training will lower the set point that your body defends, and it's absolutely crucial to preventing weight regain."
— Ch. 9

## Set Point vs. Settling Point

The book treats these similarly, but technically:
- **Set Point**: Fixed biological weight the body defends
- **Settling Point**: Weight that results from current environment/habits

The settling point model suggests that changing environment (food availability, activity level, sleep, stress) can change where weight "settles."

## Practical Implications

1. **Expect resistance** when dieting below set point
2. **Maintenance takes effort** - body wants to regain
3. **Exercise helps** lower defended weight
4. **Time matters** - longer at new weight = more adaptation
5. **Avoid yo-yo dieting** - may raise set point
— Source: Fat Loss Forever, Ch.9, Ch.11
