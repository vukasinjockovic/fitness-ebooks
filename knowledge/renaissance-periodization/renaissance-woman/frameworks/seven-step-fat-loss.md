# Seven-Step Fat Loss Protocol

**Type:** Framework
**Status:** Solid
**Source:** Renaissance Woman, Chapter 2
**Authors:** Dr. Jen Case, Dr. Melissa Davis, Dr. Mike Israetel

## Summary

A systematic, step-by-step protocol for designing and executing a fat loss diet. The framework emphasizes iterative adjustments based on bodyweight tracking rather than rigid adherence to initial calculations.

## The Seven Steps

### Step 1: Establish Base Intake

Determine maintenance calories using the activity-adjusted table:

| Bodyweight | Light/Off | Moderate | Hard |
|------------|-----------|----------|------|
| 100 lbs | 1500 | 1935 | 2400 |
| 125 lbs | 1650 | 2185 | 2650 |
| 150 lbs | 1850 | 2435 | 2900 |
| 175 lbs | 2050 | 2785 | 3200 |
| 200 lbs | 2300 | 3135 | 3500 |

**Calibration process:**
1. Start with table value based on bodyweight and activity
2. Track weight for 1-2 weeks
3. Adjust by 250-500 calories if gaining/losing
4. Repeat until weight stabilizes

**Modifiers:**
- Taller women need more calories
- Younger women need more calories
- Genetically faster metabolisms need more

### Step 2: Generate Caloric Deficit

**Target rate:** 0.5-1.0% bodyweight per week

**Calculation:**
```
a. Choose weight loss percentage (e.g., 1.0%)
b. Multiply by bodyweight: 150 lbs × 0.01 = 1.5 lbs/week
c. Multiply by 3500 cal/lb: 1.5 × 3500 = 5250 cal/week
d. Divide by 7: 5250 / 7 = 750 cal/day deficit
```

**For beginners:** Start closer to 0.5% (more sustainable)

**Warnings:**
- Too small: Diet drags on unnecessarily
- Too large: Muscle loss, performance decline, burnout, rebound

### Step 3: Calculate Macros

**The Order of Cuts (Critical):**

1. **Keep protein high:** 1g per lb LBM (cut last, if ever)
2. **Keep carbs high initially:** 1.5g per lb LBM
3. **Cut fats first:** Until minimums reached

**Fat minimum thresholds:**
- >2 months remaining: 0.3g per lb LBM
- <2 months remaining: 0.1g per lb LBM

**Example (150 lb woman, 20% BF, 120 lb LBM):**
```
Base: 2435 cal
Protein: 120g (480 cal)
Carbs: 180g (720 cal)
Fats: 137g (1235 cal)

After 500 cal cut (from fats):
Protein: 120g
Carbs: 180g
Fats: 82g (738 cal)
Total: 1935 cal
```

**When fats hit minimum:** Start cutting carbs
**Protein cuts:** Almost never needed; if eating <600 cal/day, hire a coach

### Step 4: Track Bodyweight

**Frequency:** 2-3 times per week

**Protocol:**
- Same time (morning preferred)
- After bathroom, before eating
- Naked or same clothing
- Same scale

**Why not daily?**
- Water fluctuations of ±2% are normal
- Daily weigh-ins cause unnecessary anxiety
- Need weekly averages for true trends

**Why not weekly?**
- Single weigh-in can be skewed by water/food/salt
- Miss the trend if that day is an outlier

### Step 5: Adjust as Needed

**Three scenarios:**

**A. On Track:**
- Do nothing
- Resist urge to "speed things up"
- If working, don't change

**B. Too Fast (>1% BW/week):**
- Add 250 calories (from fats if not yet cut, carbs if cut)
- Risk of muscle loss and burnout

**C. Too Slow/Stalled:**
- Cut 250 calories per day
- From fats first, then carbs
- Expected: Metabolism slows over time

**Why weight loss slows:**
1. Thyroid downregulation
2. Reduced NEAT (unconscious movement)
3. Less tissue = lower BMR

### Step 6: Plan Diet Duration

**Three rules:**
1. Create specific weight loss goal
2. Apply max pace limit (1% BW/week)
3. Maximum 12 weeks before maintenance break

**Example calculations:**
```
Woman X: 160 lbs, wants to lose 10 lbs in 8 weeks
- Minimum rate: 10/8 = 1.25 lbs/week
- Max rate: 160 × 0.01 = 1.6 lbs/week
- Verdict: Achievable

Woman Y: 180 lbs, wants to lose 30 lbs
- 12 weeks at max pace: 1.8 lbs/week × 12 = 21.6 lbs
- Verdict: Needs 2 diet phases with maintenance break
```

### Step 7: Re-establish Metabolic & Psychological States

**Why maintenance is mandatory:**

| Issue | Cause | Solution |
|-------|-------|----------|
| Thyroid downregulation | Prolonged deficit | Isocaloric eating |
| Accumulated fatigue | High work + low fuel | Rest + more food |
| Elevated AMPk | Low carbs + high cardio | Carb restoration |
| High set point | Body "remembers" old weight | Time at new weight |
| Hunger/cravings | Physiological response | Full maintenance calories |
| Motivation loss | Monotony + fatigue | Diet break |

**Maintenance phase protocol:**
1. Add back calories to eliminate deficit immediately
2. Expect 3-6 lb water weight gain (temporary)
3. Duration: 1-3 months based on diet difficulty
4. Add 250 cal/week if weight drops (metabolism recovering)

## Troubleshooting

| Symptom | Likely Cause | Fix |
|---------|--------------|-----|
| Rapid weight spike mid-diet | Water retention (salt, carbs) | Wait 3-4 days |
| Weight loss stopped | Metabolic adaptation | Cut 250 cal |
| Extreme hunger | Deficit too aggressive | Raise calories slightly |
| Performance crash | Carbs too low | Prioritize training day carbs |
| Losing strength | Losing muscle | Slow the rate, raise protein |
