# RP Cookbooks - Master Index

## Overview
The Renaissance Periodization cookbook series provides macro-tracked recipes designed to fit RP diet templates. Author: Lori Shaw. Written with guidance from Dr. Mike Israetel and Nick Shaw.

## Books Covered

### RP 14 - The Renaissance Feast (2017)
- **Focus**: 101 additional recipes (200 total with first cookbook)
- **Categories**: Proteins (Fish, Seafood, Chicken, Turkey, Beef, Eggs), Veggies, Healthy Fats, Healthy Carbs, Complete Meals, Chocolate Splurges
- **Key Feature**: Adjustable recipe format with "Your Changes" section
- **Citation**: Shaw, L. (2017). The Renaissance Feast. Renaissance Periodization.

### RP 16 - Big Flavor, Zero Fat (2018)
- **Focus**: 50 nonfat recipes for fat loss phases
- **Use Cases**: Deep fat loss phases, higher-carb massing
- **Categories**: Chicken, Sausage/Pasta, Beef, Fish/Seafood, Egg Whites, Sides, Sauces, Breakfast, Desserts
- **Key Insight**: "When deep into fat loss and all but critically-needed fats have been cut"
- **Citation**: Shaw, L. (2018). Big Flavor, Zero Fat. Renaissance Periodization.

### RP 18 - Something Sweet (2017)
- **Focus**: 50 dessert recipes with macro tracking
- **Use Cases**: Maintenance, Massing, Special occasions
- **Categories**: Cakes/Crisps/Pies, Cookies/Bars, Breakfast/Puddings/Parfaits, Snacks, Protein/Casein recipes
- **Philosophy**: "RP is not a diet, it's a lifestyle"
- **Citation**: Shaw, L. (2017). Something Sweet. Renaissance Periodization.

## Key Nutritional Concepts

### Recipe Format
All recipes include:
1. **Ingredient list** with protein/carbs/fat per ingredient
2. **Total recipe macros**
3. **Per-serving macros** (multiple portion options)
4. **"Your Changes" section** for customization
5. **Instructions**

### Macro Counting Philosophy
- Only count "count-worthy" macros
- Skip trace amounts (herbs, small amounts per serving)
- Use net carbs (total carbs minus fiber)
- Track macros "from healthy sources" not total weight

### Phase-Specific Eating
| Phase | Cookbook Focus |
|-------|----------------|
| Cutting | RP 16 (Zero Fat) |
| Deep Cut | RP 16 (Zero Fat) |
| Maintenance | All cookbooks, including splurges |
| Massing | All cookbooks, higher carbs/treats OK |

### Nutrition Label Calculator
For scaling portions:
1. A = grams of macro you need
2. B = grams of macro per serving (from label)
3. C = weight of serving (grams)
4. **D = A / B** (your multiplier)
5. **Your portion = D × C** grams
